# Pacific Northwest Explorer

A curated collection of Pacific Northwest attractions, travel guides, and tools to help discover and explore Washington, Oregon, Idaho, and British Columbia.

## Overview

This repository provides:

- 📍 Attraction guides and destination highlights
- 🏔️ National parks, mountains, lakes, and scenic drives
- 🚶 Hiking and outdoor recreation resources
- 🏙️ City guides and local recommendations
- 🗺️ Maps, trip-planning resources, and itineraries
- 🛠️ Useful travel and outdoor exploration tools
